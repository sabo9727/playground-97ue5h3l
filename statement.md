# Welcome!

This C# template lets you get started quickly with a simple one-page playground.

```C# runnable

using System;

class Hello 
{
    static void Main() 
    {
        List<int> list = new[] {1,2,3}.ToList();
        Console.WriteLine(list);
    }
}
// }
```

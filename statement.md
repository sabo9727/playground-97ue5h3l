# Welcome!

This C# template lets you get started quickly with a simple one-page playground.

```C# runnable

using System;
using System.Collections.Generic;

class Hello 
{
    static void Main() 
    {
        int[] array = new List<int> {1, 2, 3}.ToArray();
        Console.WriteLine(array);
    }
}
// }
```
